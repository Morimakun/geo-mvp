# ジオソリューションズ Phase 2 デモ

このフォルダは、ジオソリューションズ様向けの「第二段階：日次事務作業の半自動化」デモ画面です。

## このデモの目的

初期版では、FAX帳票とSalesforce CSVを見比べ、一致・不一致・要確認を整理し、Salesforce側で修正しやすいCSVを出すところまでを対象にします。

このPhase 2デモは、その先の将来イメージとして、確認後の修正管理、問い合わせメール下書き、処理履歴管理などを画面で見せるためのものです。

## 重要な注意点

このデモは本番実装ではありません。

- 既存の照合システム本体には接続していません
- 実データは使用していません
- Salesforceを更新しません
- Gmail送信を行いません
- メール送信やSalesforce修正は、必ず人が確認する前提です
- 初期版の正式スコープではありません

## デモで見せる内容

- 日次確認ダッシュボード
- 未確認 / 要確認 / 修正済み / 問い合わせ中 / 完了 の状態表示
- 要確認一覧
- 詳細確認パネル
- 問い合わせメール下書き
- Salesforce修正用CSVの出力イメージ
- 処理履歴

## 位置づけ

このデモは、初期版の先にある拡張イメージを説明するためのものです。

最初からすべてを自動化するのではなく、まずはFAX帳票とSalesforce CSVの照合支援から始め、運用が安定してから、確認・修正・連絡・履歴管理へ段階的に広げる想定です。

## 方針

AIに任せる部分と、人が確認する部分を分けます。

- AIは、見比べ、整理、下書き作成、履歴化を支援します。
- 人は、最終判断、送信前確認、Salesforce側の修正確認を行います。
